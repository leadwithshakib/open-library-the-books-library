# Getting Started with Open Library

This guide will help you set up and run the Open Library project using Docker.

## Prerequisites

Before you begin, ensure you have the following installed:

- Git
- Docker
- Docker Compose

## Installation Steps

1. Clone the repository:

```bash
git clone https://github.com/leadwithshakib/open-library-the-books-library
```

2. Navigate to the project directory:

```bash
cd open-library-the-books-library
```

3. Start the necessary containers using Docker Compose:

```bash
docker compose up -d
```

## Stopping the Application

To stop the application, run:

```bash
docker compose down
```
